# RB 1.0.0

\# RB 1.0.0



\- First stable release of the `RB` package.

\- Implemented the Redistributed-Beta (RB) distribution for unit data.

\- Added distributional functions: `dRB()`, `pRB()`, `rRB()`, and `mRB()`.

\- Added summary measures: `meanRB()`, `varRB()`, `cvRB()`, `skewRB()`, and `kurtRB()`.

\- Added parameter estimation by moments and maximum likelihood through `fitRB\_mom()` and `fitRB\_mle()`.

\- Added model comparison tools through `compareRBmodels()`.

\- Added bootstrap goodness-of-fit procedures through `gofRB\_boot()`.

\- Included reproducibility scripts for the simulation study, graphical analyses, and real-data applications.
